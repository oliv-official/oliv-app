Controls how "Categorize Similar" decides two descriptions are the same. 100% matches only identical descriptions (case-insensitive); lower values match progressively fuzzier — catching typos, trailing spaces, and minor wording differences.

Automatic Categorization
Remembers the category you assign to a description and applies it automatically to matching transactions you add or import later. Only confident matches are categorized — anything uncertain is left alone for you to review. When matching above is set below 100%, near-identical descriptions can also auto-match.
